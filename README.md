# XC Projector

XC Projector is a Next.js + Supabase consulting operations app for:

- portfolio / program / project setup
- position-based staffing
- weekly timesheets
- project status reporting
- plan vs actual and commercial reporting

Current MVP status:

- core project, staffing, time and status workflows are implemented
- skill requirements and match hints are implemented
- edit / reassign / deactivate flows are implemented for project operations
- billing overrides are implemented
- in-app user role and scoped access management are implemented
